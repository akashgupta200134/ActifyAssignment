--React Accounts Dashboard---

-A modern dashboard application built with React that allows users to manage accounts efficiently.
-The application includes account listing, search, sorting, pagination, Excel export, and a form to add new accounts.

--Features
-View account data in a dynamic table
-Search accounts instantly
-Sort table columns
-Pagination support
-Export filtered data to Excel
-Add new accounts through a form
-Sidebar navigation with active route highlighting
-Responsive UI using Tailwind CSS
-State management using Redux Toolkit


----Tech Stack----
--Frontend
-React+Vite
-Redux Toolkit
-React Hook Form
-TanStack React Table
-Tailwind CSS
-React Router
-React Icons

##Libraries
-XLSX (Excel export)


## Installation

-Clone the repository
git clone https://github.com/yourusername/react-accounts-dashboard.git

--Navigate to the project
cd react-accounts-dashboard

--Install dependencies
npm install

--Start the development server
npm run dev

#####Scope of the Project

----Current Scope----
-Add and manage account information with a form.
-View all accounts in a sortable and searchable table.
-Filter and search accounts in real-time.
-Export account data to Excel.
-Responsive design for desktop and mobile.
-Highlight active navigation links in the sidebar.


-----Future Scope-----
-Add edit and delete functionalities for accounts.
-User authentication and role-based access.
-Advanced data analytics and visualization (charts, graphs, dashboards).
-Export data in multiple formats (PDF, CSV).
-Integrate with a backend database for persistence.
-Dark mode toggle and more theme customizations.
-Multi-language support for internationalization.
